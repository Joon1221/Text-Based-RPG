Features
--------

- Runs tests on four server types (**C++ Asynchronous**, **C++ Synchronous**, **JavaServlet** using **Apache Tomcat**, and **Node.js**).
- Tracks **ping** and **bitrate**.
- Runs tests on two scenarios (**Chat** server and **Streaming** server).
